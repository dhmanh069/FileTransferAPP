# File Transfer Application

Ứng dụng truyền file qua mạng sử dụng C#, Avalonia UI và TCP Socket.

## Chức năng

- Server Start
- Client Connect
- Hiển thị Online Users
- Chọn người nhận
- Browse File
- Send File
- Server forward file
- Client nhận file
- Save File

## Công nghệ

- C#
- .NET 10
- Avalonia UI
- TCP Socket
