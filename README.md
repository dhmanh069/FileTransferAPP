## Chức năng

### 1. Kết nối hệ thống
- Khởi động Server.
- Client kết nối đến Server.
- Đăng nhập bằng tên người dùng.
- Đăng xuất (Logout).
- Hiển thị trạng thái kết nối.

### 2. Quản lý người dùng
- Hiển thị danh sách người dùng đang trực tuyến.
- Tự động cập nhật danh sách khi có người đăng nhập hoặc đăng xuất.
- Chọn người nhận để gửi tin nhắn hoặc gửi file.

### 3. Truyền file
- Chọn một hoặc nhiều file để gửi.
- Gửi file đến người dùng được chọn.
- Hỗ trợ gửi nhiều file liên tiếp.
- Hiển thị tiến trình gửi file (Progress Bar).
- Server tiếp nhận và chuyển tiếp file đến đúng người nhận.
- Client nhận file.
- Lưu file nhận được vào máy.
- Hiển thị tên file và thời gian nhận file.

### 4. Chat
- Chat trực tiếp giữa các Client.
- Hiển thị thời gian gửi tin nhắn.
- Hiển thị lịch sử trò chuyện.
- Hỗ trợ tin nhắn tự động xóa theo thời gian (/del).
- Server chuyển tiếp tin nhắn đến đúng người nhận.

### 5. Nhật ký hệ thống
- Hiển thị lịch sử đăng nhập và đăng xuất của người dùng.
- Hiển thị lịch sử gửi file.
- Hiển thị lịch sử gửi tin nhắn.
- Ghi nhận thời gian thực của các hoạt động trên Server.

### 6. Giao diện
- Giao diện Client thân thiện, dễ sử dụng.
- Hiển thị danh sách người dùng trực tuyến.
- Hiển thị khu vực trò chuyện.
- Hiển thị tiến trình gửi file.
- Hiển thị trạng thái của ứng dụng.
